# 起飞机场选择（dep_apt）

> 上游：[../SKILL.md](../SKILL.md)。

## 主要限制：跑道长度
- 飞机起飞需要一定长度的跑道来达到起飞速度，因此规划时需要指定离场机场的**最低跑道长度**。
- 机场基本信息（含**最长跑道长度**）记录在 `NavData/earth_aptmeta.dat`。
- 过滤逻辑实为：机场的**最长跑道 ≥ 阈值**。

## 选择规则
- **随机规划**时：严格遵守默认或用户给出的跑道长度限制。
- **用户指定起飞机场**时：以用户输入为主，**忽略**跑道长度限制参数。

> 说明：跑道长度是当前最主要的限制因素之一；真实世界还会看跑道宽度/道面/重量等级、口岸等，本项目从略。
